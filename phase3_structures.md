# Phase 3 Structures

سریالایزرها و viewsetها با استفاده از DRF پیاده‌سازی شده‌اند تا API نسخهٔ v1 را فراهم کنند. همهٔ endpointها از pagination و فیلتر جستجو بهره می‌برند و مجوز پیش‌فرض به‌صورت «خواندن عمومی / نوشتن با احراز هویت» تنظیم شده است. مسیرها به صورت ماژولار در هر اپ ثبت و سپس در `core/urls.py` یکپارچه می‌شوند.
