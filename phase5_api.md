# Phase 5 API

تمام اپلیکیشن‌ها endpointهای CRUD خود را از طریق Router واحد `/api/` در دسترس قرار می‌دهند. جستجوی سراسری API نیز در `apps/search/views.py` پیاده‌سازی شده تا نتایج بازی، خبر و بررسی را در یک پاسخ یکپارچه برگرداند. مستندات Swagger/Redoc می‌توانند به‌سادگی با نصب `drf-spectacular` یا `drf-yasg` اضافه شوند.
